# Code of Conduct

Blurly should be a welcoming project for people learning, experimenting, and
shipping useful image tools.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially with beginners.
- Keep feedback specific and actionable.
- Discuss technical choices without attacking people.
- Credit others for ideas, models, examples, and fixes.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Sexualized language or imagery in project spaces.
- Publishing private information without permission.
- Repeatedly derailing issues or pull requests.
- Hostile comments about someone's skill level, background, or identity.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, and pull requests that
violate this code of conduct. Repeated or severe violations may result in a ban
from project spaces.

If you need to report a conduct issue, open a private maintainer channel when
one is available. Until then, avoid public escalation and contact the project
maintainer directly.
